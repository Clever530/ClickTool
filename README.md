# ClickTool/连点器
连点器

![image](https://github.com/Clever530/ClickTool/blob/main/gui.png)


可以进行窗口置顶 默认为置顶状态（按钮显示取消置顶）

设置毫秒数来设置连点时间间隔 默认为100ms

点击快捷键旁按钮设置快捷键 默认为`/~

点击目标按键旁按钮设置连点按键 默认为鼠标左键

重置按钮重置对应行按键
